# Plonopolis

Starter projektu gry przeglądarkowej w Next.js.

## Co jest gotowe
- logowanie i rejestracja UI
- ekran farmy
- wizualne etapy farmy: 1 / 5 / 10 / 15 / 20
- hitboxy z trybem debug
- config balansu gry
- config nasion, drzew, zwierząt i ula
- mock API logowania i rejestracji

## Jak uruchomić
```bash
npm install
npm run dev
```

## Najważniejsze strony
- `/login`
- `/register`
- `/game`

## Ważne
To jest starter pod dalszą rozbudowę. Backend z prawdziwą bazą danych i autoryzacją nie jest jeszcze podłączony.
